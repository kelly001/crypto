Установка сертификата в Windows

  1. “Пуск", в поле для поиска файлов (выделенном на скриншоте) ввести команду "mmc", нажать на Программу "mmc" правой кнопкой мыши и выбрать в контекстном меню пункт "Запуск от имени администратора"
  2. В появившемся окне программы нужно открыть настройки сертификатов компьютера. Для этого:
  • В окне "Консоль1" нажимаем на пункт меню "Файл" => "Добавить или удалить оснастку..."
  • В появившемся окне "Добавление и удаление оснасток" в левом списке ищем строку "Сертификаты", нажимаем кнопку "Добавить".
  • Далее в появившемся окне "Оснастка диспетчера сертификатов", выбираем пункт меню "Моей учетной записи пользователя" и нажимаем кнопку "Далее". Проходим все экраны диалога дальше, в конце наживаем “Готово”
  • Далее разворачиваем по + список "Сертификаты" => "Доверенные корневые центры сертификации" => Сертификаты.
  1. Теперь нужно установить корневой сертификат. Для этого:
  • Разворачиваем список "Сертификаты" => "Доверенные корневые центры сертификации" => Сертификаты
  • Нажимаем правой кнопкой по “Сертификаты”, “Все задачи” - “импорт”
  • Следуем указаниям в диалоговом окне, выбирать сертификат с именем, указанным при генерации корневого сертификата .cer
  • В ответ на предупреждение системы нажимаем “установить”, в конце “Готово”
  1. Устанавливаем личный сертификат пользователя. Для этого:
  • Разворачиваем список "Сертификаты" => "Личное" => Сертификаты
  • Нажимаем правой кнопкой по “Сертификаты”, “Все задачи” - “импорт”
  • Следуем указаниям в диалоговом окне, выбирать сертификат с именем, указанным при персонального сертификата .p12
  • Нужно будет ввести пароль для личного сертификата ( последний из вводимых в консоли)
  1. После установки сертификатов файлы сертификатов сохранить в надежном шифруемом хранилище, с компьютера сами файлы можно удалить.

Подпись документов личным сертификатом

Теперь в программах MS Office можно будет устанавливать личную подпись.

Для этого нужно сохранить документ, потом выбрать в меню Сервис - Параметры - Безопасность - Цифровые подписи и выбрать из списка сертификатов нужный.

Для того, чтобы  сертификат был действителен и доступен,  необходимо:

  • Уровень безопасности пользователей в Windows - низкий
