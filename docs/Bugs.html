Технические особенности и рекомендации

Версия программы 1.0

  • После создания нового сертификата компании нужно обновить главное окно (Меню -> главная)
  • После регистрации организации идет авторизация пользователя в программе сразу - окно авторизации закрываем сами
  • Удаление пользователей - удаляется первый по id, нет возможности выбора. При удалении пользователя у него обновляется статус (поле status) на false. Сам пользователь остается в системе.
  • Отзыв сертификата -  обновляется статус (поле status) на false, не отображается больше в интерфейсе, можно создавать новый. Сам сертификат также создается в системе.
  • Поле timestamp - дата последнего обновления записи.
