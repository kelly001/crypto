Инструкция для пользователей

Установка, настройка сервера

Для работы программы на компьютере должны быть:

  • Установленная последняя версия Java
  • Настроенный сервер MySQL

В базу данных нужно импортировать настройки базы данных и пользователя для программы. Файлы с настройками и дампом прилагаются, в директории программы. Файлы: crypto_db.sql

Регистрация компании

При первом запуске нужно зарегистрировать новую организацию. Для этого нажмите кнопку “Регистрация” в окне авторизации. После создания организации авторизуйтесь с введенным email и паролем, откроется главное окно программы.

Сначала создайте сертификат компании. Кнопка “Добавить новый” под “Сертификатами пользователя”.

Создание нового сертификата.

Аккуратно введите все значения полей латинницей (!). Особое внимание на:

Имя пользователя - так будет отображаться пользователь во всех подписанных документах.

Имя файла - по этому имени будут сохранены файлы сертификата пользователя, ключей пользователей и организации.

Сертификат компании: “RootCertificate”; ключи: "filename"+"RootPublicKey", "filename"+"RootPrivateKey"

Сертификаты пользователя: “Имя файла” - которое вы ввели; ключи: "filename" + "PublicKey", "filename" + "PrivateKey".

После подтверждения (нажатие кнопки “создать”) дождитесь создания сертификата. Будет диалог, где нужно ввести пароль для контейнера сертификата. Минимальная длина пароля - 6 символов. Сертификат PKS (для импорта в Windows и браузеры) сохраняется под именем “Имя пользователя” - которое указали при создании сертификата + “.p12”

Создание/добавление сотрудников

В меню можно добавлять/удалять сотрудников в компанию.

Чтобы добавить нового сотрудника: Редактирование -> Сотрудники->Добавить

После этого он будет отображаться в списке сотрудников компании ( Просмотр -> сотрудники). Из окна со списком сотрудников можно создавать сертификаты для сотрудников компании. Процедура аналогична, как при создании сертификата компании.
